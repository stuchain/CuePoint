# Export Control — CuePoint

**Version 1.0 — 2026-02-03**  
**Last updated**: 2026-02-03

## Overview

CuePoint is open-source software. This document describes our distribution policy and export control considerations for international distribution.

## Distribution

- CuePoint is distributed publicly via GitHub and release assets.
- No intentional restrictions are placed on distribution by geography.
- Users are responsible for compliance with their local laws.

## Export Control Compliance

- CuePoint does not contain encryption or controlled technology beyond standard open-source libraries.
- Dependencies are common Python packages (e.g., requests, PySide6) with permissive licenses.
- If you are in a jurisdiction that restricts software import/export, please verify compliance with your local regulations before downloading or using CuePoint.

## Embargoed Regions

We do not actively block access from any region. However, users in embargoed or restricted regions may be subject to GitHub's or other service providers' terms of service. Please ensure your use complies with applicable law.

## Contact

For questions about export control or distribution, open an issue in the CuePoint repository.
