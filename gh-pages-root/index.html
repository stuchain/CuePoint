About CuePoint

Match your Rekordbox tracks to Beatport and get key, BPM, label, genre, and release date without manual lookup.

Export from Rekordbox → run CuePoint → review or re-import. Full audit trail for every match.

CuePoint includes inKey (Beatport metadata for Rekordbox playlists) and inCrate (inventory, charts, new releases, and Beatport playlists from your collection).

Why CuePoint

Clean metadata, traceable decisions, and built to handle large libraries.

Clean metadata

Key, BPM, label, genre, release date: consistent and reviewable.

Auditable matches

Every query and candidate is logged so you can verify decisions.

Built for scale

Handles large libraries with parallel search and configurable time budgets.

How it works

Three steps from library export to updated metadata.

  • 1Export playlists from Rekordbox as XML.
  • 2Open them in CuePoint and match tracks to Beatport.
  • 3Review results and re-import or apply metadata in Rekordbox.
