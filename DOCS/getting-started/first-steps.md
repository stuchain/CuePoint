# First Steps

Welcome to CuePoint! This guide will help you get the most out of the application.

## Understanding the Interface

### Main Window

- **Track List**: Shows all imported tracks
- **Status Bar**: Displays processing progress
- **Menu Bar**: Access all features

### Key Features

- **Import**: Load Rekordbox XML files
- **Process**: Enrich tracks with Beatport metadata
- **Export**: Save results in various formats
- **Filter**: Find specific tracks quickly

## Workflow Best Practices

1. **Start Small**: Test with a small XML file first
2. **Review Matches**: Always review match scores
3. **Export Regularly**: Save your work frequently
4. **Use Filters**: Filter tracks to find what you need

## Common Tasks

### Finding a Specific Track

1. Use the search/filter box
2. Type track name or artist
3. Results update in real-time

### Checking Match Quality

1. Look at the match score column
2. Scores above 80% are usually reliable
3. Review track details to verify

### Exporting Results

1. Select tracks (or leave all selected)
2. Choose export format
3. Select destination
4. Click Export
